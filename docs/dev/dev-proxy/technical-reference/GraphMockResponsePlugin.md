# GraphMockResponsePlugin

Mocks responses sent in batch requests to Microsoft Graph.

## Plugin instance definition

```json
{
  "name": "GraphMockResponsePlugin",
  "enabled": true,
  "pluginPath": "plugins\\dev-proxy-plugins.dll",
  "configSection": "mocksPlugin"
}
```
